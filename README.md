# Crypto Market Intelligence Platform

An AI-assisted market intelligence and decision-support platform for cryptocurrency markets.

> **Architecture focus:** Clean data pipeline · SMA analytics · rule-based AI interpretation ·
> cloud-ready deployment · explainable design decisions.

---

## What it does

| Tab | Description |
|---|---|
| **Scanner** | Scans top 30 liquid USDT pairs, ranks by SMA signal (BUY / HOLD / SELL) |
| **Chart** | Price chart with SMA20/SMA50 overlays, signal banner, AI insight panel |
| **Backtest** | Walk-forward simulation vs buy-and-hold, trade log, win rate |

## Signal logic

```
BUY  = Price > SMA20 > SMA50   (uptrend alignment)
SELL = Price < SMA20 < SMA50   (downtrend alignment)
HOLD = anything else            (consolidation / transition)
```

Intentionally simple. The architecture is the story.

## Running locally

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

- API available at `http://localhost:8000`
- Auto-generated docs: `http://localhost:8000/docs`
- Health check: `http://localhost:8000/health`

### Frontend

Open `frontend/index.html` directly in a browser, or serve it:

```bash
cd frontend
python -m http.server 3000
# then open http://localhost:3000
```

The frontend calls `http://localhost:8000` by default. To point it at a different backend, set `window.API_BASE` before `app.js` loads:

```html
<script>window.API_BASE = "https://your-backend.azurewebsites.net";</script>
```

## API endpoints

| Method | Endpoint | Description |
|---|---|---|
| `GET` | `/health` | Liveness probe |

## AI insight layer — the LLM seam

`insight.py` has one public function:

```python
def generate(base, timeframe, price, change_24h, signal) -> str:
    ...
```

Today: rule-based NLG (deterministic, free, testable).

To swap in an LLM — replace only the function body:

```python
async def generate(...) -> str:
    response = await openai.chat.completions.create(
        model="gpt-4o",
        messages=[{"role": "system", "content": SYSTEM_PROMPT},
                  {"role": "user",   "content": json.dumps(signal)}]
    )
    return response.choices[0].message.content
```

The route, API endpoint, and frontend are **unchanged**.
